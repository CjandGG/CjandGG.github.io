Pandas

Python数据分析库(Python data analysis libs)

Pandas是一个开源的，BSD许可的库，为Python (opens new window)编程语言提供高性能，易于使用的数据结构和数据分析工具。

Pandas是NumFOCUS (opens new window)赞助的项目。这将有助于确保Pandas成为世界级开源项目的成功，并有可能捐赠 (opens new window)给该项目。

Pandas 是 Python 的核心数据分析支持库，拥有快速、灵活、明确的数据结构，旨在简单、直观、快速地处理关系型、标记型数据，是一款强大、灵活的开源数据分析工具。

Pandas 主要数据结构是一维数据(Series)、二维数据（DataFrame），这两种数据结构能满足金融、统计、社会科学等领域中大多典型用例。Pandas 是基于 NumPy 开发，可以与其它第三方计算支持库完美集成。

方便的数据视图化、直观地查看数据情况是非常必要的，对于非专业的数据分析人员查看数据走势、分布是一种很好展现形式。数据可视化相关的库也有很多，如 Matplotlib、Seaborn、Altair 等都可以快速学习并实现。

Python在数据处理和准备方面一直做得很好，但在数据分析和建模方面就没那么好了。Pandas帮助填补了这一空白，使您能够在Python中执行整个数据分析工作流程，而不必切换到更特定于领域的语言，如R。

数据结构二 : DataFrame

索引

类型 表现方式 作用
中括号 [“关联列名称”] 只能获取列
属性 df.colname.indname 顺序获取列和行
关联获取 df.loc[y:x] 只能获取关联型索引
枚举获取 df.iloc[y:x] 只能获取枚举型索引

pd的运算

  • Series在index相同的条件下可以相互运算
  • DataFrame在index和column相同的情况下可以运算(一般不做矩阵的点积)

CURD

空值检测和填充删除

检测空数据

  • isnull()
  • notnull()
  • Pandas中主要是检测每一列的空值

空值删除

  • dropna(axis=0|1)
  • 优先删行,如果列空值比例过大,删列

填充空值

  • fillna()
  • values : 指定填充一个固定的值
  • method : 后补和前补

索引的堆变换

  • stack() : 列索引转变为行索引
  • unstack() : 行索引转变为列索引(透视表变换)

DF拼接

  • concat
  • []当中可以写多个df
  • 行索引会重复,ignore_index重新设定索引
  • axis切换合并的方向(axis=1很少见)
  • join=”outer”
  • 一般是当做union取使用的

append

df.append(df|sr,ignore_index=True)

  • 没有axis,只能实现上下合并

DF合并

  • merge

  • 每次只能合并两组数据

  • 要求列名相同且数据类型相同

  • 假如列名不同,使用left_on,right_on

  • 假如多个列名相同on,指定关联列名,suffixes修改无关同名列的后缀(防止冲突)

  • left_index和right_index在老版本中代表使用行索引作为关联(新版本中无所谓)

join

df.join(df1,on='col')

  • 要求合并的两组数据关联列名相同

映射

  • map

    • 只能够被Series所调用
    • 可以传递dict和func

  • apply

    • 可以被df或series调用
    • 如果被df调用,需要控制方向,axis=1,行方向控制
    • 如果被df调用,那么计算速度会大幅度下降
    • 和transform非常类型

  • agg

    • 不同的列实现不同的聚合计算
    • 支持dict{“cols1”:”mean”,”cols2”:”min”}

  • rename

    • 主要是修改列名称的
    • 支持dict类型{“old_col_name”:”new_col_name”}

  • replace

    • 替换Series当中的元素

去重

  • duplicated
    • subset : 指定的列对比重复值
    • keep : 重复值的起始位置
  • drop_duplicats
    • subset : 指定的列对比重复值

描述统计

df.describe()

  • 返回每一列的均值
  • 返回每一列的标准差
  • 最大和最小
  • 返回数量
  • 返回四分位

排序

  • sort_values(by=[cols])

  • sort_index(ascendeing=True)

分组

  • groupby(by=[cols])
    • 返回DataFrameGroupby对象,当前对象只能调用映射函数或者是聚合函数
  • groupby(by=[cols]).col
    • 返回SeriesGroupby对象,当前对象只能调用映射函数或者是聚合函数

时间操作

pd.to_datetime(series,cache=True)

  • 把字符串的时间转变为datetime64类型

df.to_peroid()

  • 要求时间序列必须作为行索引
  • 时间范围的映射
  • 时间计算的时候不可用(画图的时候不能用)

Series.dt

  • Series必须是时间类型

df.resample("D")

  • 数据降帧的时候使用的
  • 数据聚合的时候，代替groupby进行使用

时间计算

  • (datetime64 - datetime64) // np.timedelta64(1, ‘D’)
